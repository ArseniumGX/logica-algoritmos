# Exercício Lógica de Programação
## Básico ao avançado
Repositório com exercícios em logica de programação para exercitar a lógica e habilidades em linguagens de programação. Pode ser feito em qualquer linguagem de programação.
Aqui fiz o uso do javascript (NodeJS) por ser minha stack principal. Toda resolução e códigos são de minha autoria, mas em alguns casos uso o resultado experado pelo autor por questões de economia de tempo.
